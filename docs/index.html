6º Projeto ABP – MindCare: Aplicativo de Bem-Estar com Análise Emocional via IA

Desenvolvimento de um aplicativo mobile de bem-estar emocional capaz de analisar o humor do usuário através de processamento de linguagem natural (NLP) com IA preditiva. O sistema sugere trilhas de atividades de 5 a 7 minutos por 7 dias, baseadas na emoção detectada — como raiva, ansiedade, tristeza ou estresse. Atuei no desenvolvimento do frontend em React Native com TypeScript, criação das telas completas no Figma, integração com o backend Node.js + MongoDB e construção de um micro-serviço adicional em Python (FastAPI) dedicado à classificação de emoções. Além disso, implementei a navegação entre telas, lógica de trilhas, registro de progresso, gráfico de humor, autenticação e integração das APIs.
