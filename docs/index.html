Projetos Acadêmicos


1º Projeto ABP - Aplicação Web Instruções SCRUM


Desenvolvimento de uma aplicação web para instruções sobre metodologias ágeis, com foco no SCRUM. Fui responsável pela criação do protótipo no Figma, pela elaboração do conteúdo explicativo sobre papéis e eventos do SCRUM, e pela implementação da navegação entre páginas utilizando HTML, CSS e JavaScript.



GitHub Repository

2º Projeto ABP - Aplicação Web para Aluguel de Bicicletas Peer-to-Peer


Desenvolvimento de uma plataforma web para aluguel de bicicletas no modelo peer-to-peer. Fui responsável pela criação do protótipo no Figma, pela modelagem do banco de dados PostgreSQL e pelo desenvolvimento do módulo de cadastro de usuários utilizando TypeScript e React.



GitHub Repository

3º Projeto ABP - Sistema Web para Gerenciamento de Projetos


Criação de um sistema web para controle de tarefas, progresso e análise de dados em projetos. Fui responsável pela criação do protótipo no Figma, pela construção das páginas de dashboard com gráficos em React e pela integração dos dados utilizando MongoDB.



GitHub Repository

4º Projeto ABP - Aplicação Mobile para Controle de Alimentação e Nutrição


Desenvolvimento de um aplicativo mobile para registro e monitoramento de ingestão de alimentos e nutrientes. Fui responsável pela criação do protótipo no Figma, pelo desenvolvimento da interface de registro de alimentos no React Native e pela configuração da conexão com os bancos de dados MongoDB e PostgreSQL.



GitHub Repository

5º Projeto ABP - D-FireTrack: Sistema de Monitoramento de Queimadas


Desenvolvimento de um sistema web para monitoramento de queimadas e cicatrizes de fogo através de dados geoespaciais. Fui responsável pela criação do protótipo no Figma, pela implementação da integração entre o mapa interativo e a seleção de coordenadas, bem como pelo desenvolvimento do painel de filtros por período de datas e área geográfica utilizando React e TypeScript. Além disso, colaborei na estruturação do backend para a gestão das informações no banco de dados MongoDB.



GitHub Repository
